    • O que são Clients?

    Voltar

    É o dispositivo que está consumindo aquela informação na internet, uma entidade que consome os serviços de uma outra entidade servidora, em geral através do uso de uma rede de computadores numa arquitetura cliente-servidor.

    Cliente no Linguajar Web

    Voltar

    Basicamente, quando é feito um acesso em um site, temos o acesso na internet e o servidor nos retorna arquivos HTML, CSS, Javascript, imagens e etc.

    Importe ressaltar que a velocidade do servidor (internet) do cliente vai influenciar no tempo de retorno desses arquivos.

    Navegadores

    Voltar

    Navegadores são programas criados por empresas, utilizados para abrir/executar documentos, também chamdos de browsers .

    Hoje possuimos uma série de opções de navegadores, sendo todos gratuitos.

    Aplicações Web

    Voltar

    São soluções criadas que possuem a internet como meio de comunicação entre o Client x Server, não sendo necessário a sua instalação.

    Por exemplo, temos o Facebook quando utilizamos o navegador para acessa-lo.

    Dispositivos móveis

    Voltar

    Os dispositivos móveis são os computadores de bolso. É possível realizar todas as tarefas que desenvolvemos no computador comum, mas com o diferencial da mobilidade.

    Podemos citar alguns exemplos de dispositivos móveis: smartphone, ultrabook, notebook, netbook, GPS, tablet.
